# System Map

- `gamekit-feature-analyst` defines feature boundaries and acceptance criteria.
- `gamekit-placeholder-artist` prepares graybox resources.
- `gamekit-tech-art-contracts` defines replacement-safe Asset ABI contracts.
- `gamekit-gameplay-engineer` implements gameplay against the approved spec and ABI.
- Engine integrators capture platform-specific connection and verification constraints.
- `gamekit-qa-verifier` owns QA case authoring and verification evidence.
