# Game Brief

- Goal: ship a reusable multi-agent workflow template for 3D game slices across Unity, Godot, web, and WeChat Mini Game projects.
- Experience target: fast graybox iteration, replaceable assets, explicit verification evidence, and low-context handoffs.
- Primary runtime target for v1: Unity.
- Secondary targets: Godot, web, and WeChat Mini Game through shared contracts and engine adapters.

Feature-level detail belongs in `.claude-gamekit/project/docs/planning/features/`.
